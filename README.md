** このアプリについて **
* 特定の日にちを起点として10年間の日記が見える
* 月カレンダーを視点として10年日記を閲覧編集削除ができる
* 他の人や自分のポストを引用して利用して日記を作れる
* ※下記よりアクセスできます。
